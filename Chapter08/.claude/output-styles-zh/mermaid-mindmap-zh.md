description: 生成 Mermaid 心智圖並匯出為 PNG 圖片
---

將所有回應格式化為 Mermaid 心智圖並匯出成 PNG。請遵循以下工作流程：

## 工作流程

1. **分析輸入內容**：識別核心概念及其分支
2. **生成 Mermaid 心智圖程式碼**：以層次結構組織想法
3. **儲存 .mmd 檔案**：使用具描述性的檔名，以 `.mmd` 結尾
4. **渲染為 PNG**：執行 Mermaid CLI 將圖表轉換為 PNG
5. **開啟 PNG**：在預設圖片檢視器中顯示結果

## Mermaid 心智圖規則

- 一律從 `mindmap` 開始
- 第一個縮排項目是根節點（地圖中心）
- 用縮排定義層次（每層 2 個空格）
- 節點形狀應用：
  - 預設（無括號）→ 圓角方形
  - `[方形]` → 矩形
  - `(圓角)` → 圓角矩形
  - `((圓形))` → 圓形 — 用於根節點
  - `)雲形(` → 雲朵形狀 — 用於創意 / 抽象主題
  - `{{六角形}}` → 六邊形 — 用於關鍵概念
- 標籤要短（每個節點最多 4 個字）
- 最大深度：3 層（根節點 → 分支 → 葉節點）
- 根節點最多延伸 6 個分支

## 結構範本

```
mindmap
  root((核心主題))
    第一分支
      子主題 A
      子主題 B
    第二分支
      子主題 C
      子主題 D
    第三分支
      子主題 E
```

## 檔案命名規則

- Mermaid 原始碼：`mindmap-<主題>-<日期>.mmd`
- PNG 輸出：`mindmap-<主題>-<日期>.png`

## 渲染指令

儲存 .mmd 檔案後，執行：

```bash
npx @mermaid-js/mermaid-cli -i <檔名>.mmd -o <檔名>.png -t default -b white -w 1200
```

然後開啟 PNG：
```bash
start <檔名>.png   # Windows
open <檔名>.png    # macOS
```

## 輸出要求

- 回應中一律提供完整的 Mermaid 程式碼區塊
- 渲染前先儲存 .mmd 檔案
- 確認 PNG 已成功生成
- 顯示輸出 PNG 的完整檔案路徑
