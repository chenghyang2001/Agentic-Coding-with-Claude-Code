這是一個使用 [`create-next-app`](https://nextjs.org/docs/app/api-reference/cli/create-next-app) 建立的 [Next.js](https://nextjs.org) 專案。

## 快速開始

首先，啟動開發伺服器：

```bash
npm run dev
# 或
yarn dev
# 或
pnpm dev
# 或
bun dev
```

使用瀏覽器開啟 [http://localhost:3000](http://localhost:3000) 查看結果。

你可以透過修改 `app/page.tsx` 來編輯頁面，檔案儲存後頁面會自動更新。

本專案使用 [`next/font`](https://nextjs.org/docs/app/building-your-application/optimizing/fonts) 自動優化並載入 [Geist](https://vercel.com/font)，這是 Vercel 的全新字型家族。

## 深入學習

若想深入了解 Next.js，請參考以下資源：

- [Next.js 官方文件](https://nextjs.org/docs) — 了解 Next.js 的功能和 API
- [學習 Next.js](https://nextjs.org/learn) — 互動式 Next.js 教學

你也可以查看 [Next.js 的 GitHub 儲存庫](https://github.com/vercel/next.js)，歡迎提供意見和貢獻！

## 部署到 Vercel

部署 Next.js 應用程式最簡單的方式，是使用 Next.js 創作者推出的 [Vercel 平台](https://vercel.com/new?utm_medium=default-template&filter=next.js&utm_source=create-next-app&utm_campaign=create-next-app-readme)。

詳情請參閱 [Next.js 部署文件](https://nextjs.org/docs/app/building-your-application/deploying)。
